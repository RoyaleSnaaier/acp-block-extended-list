== Description ==
ACP Block Extended List is a custom block plugin for creating extended ordered lists in WordPress. This plugin provides additional styling and functionality for ordered lists.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/acp-block-extended-list` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the block editor to add the "Extended Ordered List" block to your posts or pages.
